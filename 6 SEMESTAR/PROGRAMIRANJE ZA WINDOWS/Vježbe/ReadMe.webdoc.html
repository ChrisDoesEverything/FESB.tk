U ovom folderu nalaze se rješenja vježbi sa Programiranja za Windows. Svaka vježba je zapakirana u svoju zip arhivu koja zavisno od vježbe sadrži jedan ili više foldera s projektima. Projekti su nazivani po jednostavnoj konvenciji proj_ + redni broj projekta... Broj projekata po vježbi je minimalan, sve što može stati u isti izvorni kod stavljeno je zajedno, ali neke vježbe jednostavno traže više projekata unutar solutiona...

Svi primjeri su napravljeni u besplatnom IDEu - SharpDevelop (http://www.icsharpcode.net/OpenSource/SD/) koji toplo preporučam i to ne kao alternativu Visual Studiu, nego kao nešto što je još bolje od njega... S druge strane ljudi koji stoje iza SharpDevelopa autori su i ILSpy-a (http://wiki.sharpdevelop.net/ilspy.ashx) i ako vas zanima bacite oko i na ovaj maleni programčić čija je osnovna svrha dekompajliranje .Net assembly-a odnosno od izvršne datoteke natrag do izvornog koda...

Svejedno, ako radite i u VSu, VS Express-u ili nečem sličnom ne bi trebali imati problema u otvaranju i pokretanju ovih programa, a ako opet negdje i zapnete javite se... Za svaki slučaj sa kodom dolaze i prekompajlirani assembly-i (.exe datoteke) i to release i debug verzije, koji se nalaze u podfolderima bin/Debug odnosno bin/Release.

Jedini ozbiljniji problem s kojim se možete susresti je razlika u target frameworku odnosno činjenica da sam propustio voditi računa o tome da mi svi projekti imaju istu ciljanu verziju s obzirom da osobno imam sve verzije .Net frameworka instalirane na svom računalu... Nisam provjeravao, ali mislim da je sve podešeno na 3.0 i više, ali ako kome treba baš neka specifična verzija neka se slobodno javi i staviti ću je na net skupa s originalnim uploadom...

U svakom slučaju nadam se da će vam kodovi biti od koristi i kao i obično stavljam ih na web bez ikakvih ograničenja uz iskreni apel da se potrudite makar izmijeniti dizajn formi kako bi poštedili i sebe i mene eventualnih neugodnosti.
